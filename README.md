This project presents an interactive data visualization dashboard built using Tableau to analyze an orders dataset from various cities across India. The objective of the project is to explore and visualize order patterns, city-wise performance, and overall sales trends using effective data visualization techniques.

The dashboard provides insights into how orders are distributed across different cities and helps identify regions with higher order volumes. Various visual elements such as bar charts, maps, and trend graphs are used to represent the data in a clear and interactive manner. Filters and interactive components allow users to explore the dataset from multiple perspectives and better understand regional order behavior.

Through this project, important insights about customer demand and order distribution across cities can be easily identified. The visualization helps in understanding patterns that may support better business decisions, such as identifying high-performing cities or observing variations in order activity.

This project demonstrates practical skills in data analysis, dashboard design, and data visualization, and highlights how visual analytics tools can transform raw datasets into meaningful and actionable insights.
